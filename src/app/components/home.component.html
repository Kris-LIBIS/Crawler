Usage:

Use the 'Single' page to lookup a single object. Each additional query will create an extra entry in the list. Switching to the Home page or the 'Range' page will clear the list.

Use the 'Range' page to lookup a range of objects.

In the results list, click on the 'CSV' button to download the results as a comma-separated file.

Right click on any entry in the result list for a context menu. 'Preview' to see the digital document, 'Object' to open the document viewer, 'Alma' to open the Alma ID in Alma. The 'Delete' option removes the selected row from the list.

Clicking on the PID number will also open a new window with the document viewer.
